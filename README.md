# Astra Uran

## Описание
**Astra Uran** — это мини-проект, интегрированный с API OpenAI для анализа параметров давления и потока в системе транспортировки урана.

---

## Функциональность
- Анализ параметров давления и потока транспортной системы.
- Интеграция с OpenAI API для интеллектуальной обработки данных.
- Простая и интуитивно понятная структура.

---

## Используемые технологии
Проект написан с использованием следующих технологий:
- **TypeScript** (80.9%): Основной язык разработки, обеспечивающий строгую типизацию и масштабируемость кода.
- **CSS** (11.3%): Используется для стилизации пользовательского интерфейса.
- **JavaScript** (6.9%): Применяется для реализации дополнительных интерактивных возможностей.
- **HTML** (0.9%): Основная структура веб-приложения.

## Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Ansar-Askerbek/Astra-Uran.git
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd Astra-Uran
   ```

3. Установите зависимости:
   ```bash
   npm install
   ```

4. Запустите приложение в режиме разработки:
   ```bash
   npm run dev
   ```

---

## Системные требования
- Node.js 14.x или выше.
- NPM 6.x или выше.
- Доступ к интернету для взаимодействия с OpenAI API.
